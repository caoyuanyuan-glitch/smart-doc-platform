# 智能技术文档平台 v1.0.1 更新日志

**版本**: v1.0.1  
**日期**: 2026-06-10  
**更新人**: TRAE SOLO 开发团队

---

## 本次更新内容

### 1. 左侧菜单栏层级优化 ✅
- **重构结构**：6大应用作为第一层菜单（文档审核、智能润色、智能问答、内容生成、文档对比、格式转换）
- **子页面级联**：每个应用下添加了子页面菜单
- **折叠功能**：侧边栏支持折叠展开

### 2. 文件上传失败修复 ✅
- 添加更完善的错误处理和详细的错误信息提示
- 文件类型验证（支持IDML格式）
- 文件大小限制（最大50MB）

### 3. 智能润色功能完善 ✅（FR-002）
- **润色范围选择**：单段润色 / 整章润色 / 全文润色
- **并排对比视图**：
  - 左侧原文（红色删除线标记）
  - 右侧润色建议（绿色高亮标记）
- **操作按钮**：确认修改、保留原文、重新润色、复制结果、导出文档
- **统计信息**：修改字数、优化句子数、处理时间
- **文档导入**：支持从文件导入内容进行润色
- **导出功能**：支持TXT、Markdown格式导出

### 4. 智能问答改进 ✅（FR-003）
- **预设知识库**：支持技术文档、历史经验、产品说明书、系统架构、运维文档、安全文档
- **无需上传文件**：可直接基于预设知识库进行问答
- **快捷问题**：提供常见问题快速入口
- **双模式支持**：预设知识库和上传文档两种问答模式

### 5. 文档对比IDML支持 ✅（FR-005）
- 新增IDML格式解析器
- 前端上传组件支持`.idml`格式
- 添加IDML文件类型识别

## 运行说明

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 后端启动
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### 访问地址
- 前端：http://localhost:5173
- 后端：http://localhost:8000
